# 🚀 NeuroMotion

### Gesture-Controlled 3D Particle Engine

------------------------------------------------------------------------

## 🧠 Overview

**NeuroMotion** is a real-time, gesture-controlled 3D particle system
built using **Three.js** and **MediaPipe**. It allows users to interact
with a dynamic particle environment using natural hand gestures captured
via a webcam.

This project explores the intersection of **computer vision**, **WebGL
rendering**, and **human-computer interaction**, enabling intuitive,
touchless control of complex visual systems directly in the browser.

------------------------------------------------------------------------

## ✨ Features

-   🎯 Real-time hand tracking using MediaPipe\
-   🌌 Interactive 3D particle system (\~15,000 particles)\
-   🖐 Gesture-based controls:
    -   Open Hand → Rotate & change colors\
    -   Pinch → Expand / contract particles\
    -   Fist (hold) → Switch particle shapes\
-   🔄 Smooth morphing between shapes:
    -   Sphere, Heart, Saturn, Flower, DNA\
-   🎥 Floating live camera preview\
-   ⚡ Optimized for high-performance desktop rendering

------------------------------------------------------------------------

## ⚙️ Installation

``` bash
git clone https://github.com/your-username/neuromotion.git
cd neuromotion
```

Run locally:

``` bash
open index.html
```

Recommended (local server):

``` bash
npx serve
```
